## Setup Environment - Shell/Terminal
Pastikan Anda sudah berada di dalam folder proyek ini, lalu jalankan perintah berikut:

mkdir proyek_analisis_data
cd proyek_analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt

## Run Streamlit App
Untuk menjalankan dashboard secara lokal, gunakan perintah berikut di terminal:

streamlit run dashboard/dashboard.py
